# 💰 Cash Track - CS50 Final Project

This is my final project for **Harvard's CS50x**. It's a web-based financial management application built with **Python**, **Flask**, and **SQL**.

## 🚀 Features
* **User Authentication**: Secure login and registration.
* **Transaction Management**: Add, view, and delete expenses or income.
* **Live Balance**: Automatically updates your cash balance after every transaction.
* **Responsive Design**: Works on mobile and desktop using Bootstrap.

## 🛠️ Tech Stack
* **Backend**: Flask (Python)
* **Database**: SQLite
* **Frontend**: HTML5, CSS3 (Bootstrap)

## 💻 How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the app: `flask run`.
